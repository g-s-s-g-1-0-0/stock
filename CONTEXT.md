# 투자 전략 시스템 컨텍스트

## 목적
- 한국/미국 주식을 같은 프레임으로 스캔하되, 장기 추세와 시장 상태에 따라 진입 성격을 분리한다.
- 기본 흐름은 `관망 → 매수 → 관망/매도 → 신규 진입 제한`이다.
- 이 문서는 실전 운용의 핵심 매매 기준과 이유만 정리한 기준서이며, 구현 세부는 포함하지 않는다.

## 전략 철학
- `MA200 위`에서는 장기 우상향 안에서 움직인다고 보고, 눌림 매수와 모멘텀 재가속을 구분해 진입한다.
- `MA200 아래`에서는 추세 이탈 상태로 보고, 공포 구간의 기술적 반등만 제한적으로 노린다.
- 나스닥 기준값은 QQQ로 보고, VIX는 공포장 반등(B그룹)과 일부 매크로 참고에 사용한다.
- 고점 청산은 VIX 급등보다 QQQ의 장기 이격도와 일봉/주봉 RSI 과열 후 꺾임을 우선 본다.
- 주요 이벤트 당일에는 신규 진입만 막고, 기존 보유 포지션의 매도 판단은 정상적으로 수행한다.

## 1. 시장 필터

### 이벤트 당일
- 신규 진입 / 추가 진입 차단
- 기존 보유 포지션은 매도 조건 충족 시 정상 매도
- 기존 보유 포지션이 보유 유지 조건 충족 시 `매수` 유지
- 기존 보유 포지션이 보유 유지 조건을 잃으면 `관망` 전환 가능

### 나스닥 고점 청산
- 먼저 QQQ 시장 국면을 나눈다.
  - **급락 후 회복장**: 최근 60거래일 QQQ 200일선 이격도 최저값 `≤ -5%`이고, 현재 이격도 `≥ 0%`
  - **비회복장/고점 횡보장**: 위 조건이 아닌 상태
- 회복장 기준:
  - QQQ 이격도 `> +22%`, 또는
  - QQQ 이격도 `> +18%` + 주봉 RSI(14) `≥ 65` + 일봉 RSI(14) `≥ 65` + 일봉 RSI 하락 + QQQ MACD Histogram `D < D-1 < D-2`
- 비회복장 기준:
  - QQQ 이격도 `> +16%` + 주봉 RSI(14) `≥ 65` + 일봉 RSI(14) `≥ 65` + 일봉 RSI 하락, 또는
  - QQQ 이격도 `> +14%` + 주봉 RSI(14) `≥ 65` + 일봉 RSI(14) `≥ 65` + 일봉 RSI 하락 + QQQ MACD Histogram `D < D-1 < D-2`

영향:
- 강제청산 적용 대상: **B / D** 보유분
- 강제청산 제외 대상: **A / C / E / F** 보유분
- 신규 진입 / 추가 진입은 전략과 무관하게 전부 차단
- 메인 포지션과 추가매수 슬롯도 각 lot의 전략 기준으로 B/D만 청산하고 A/C/E/F는 유지
- 48시간이 지나도 고점 청산 상태가 유지되면 계속 관망

이유:
- 급락 후 회복장의 `+14%`는 회복 랠리 중간일 수 있으므로 상단을 넓게 본다. 반대로 회복장이 아닌 고점 횡보장에서는 RSI 둔화와 MACD 둔화를 함께 확인해 하루 흔들림에 너무 빨리 청산되는 일을 줄인다.

### 나스닥 하락장 필터
- 적용 대상: **A / C / D / E / F**
- **B그룹은 예외**
- QQQ 이격도 = `(QQQ 현재가 / QQQ MA200 - 1) × 100`

| 이격도 | 판단 |
|--------|------|
| `≥ -2.5%` | 차단 해제 |
| `-12% 초과 ~ -3% 미만` | 하락장 필터 차단 |
| `-3% ~ -2.5%` | 히스테리시스 유지 구간 |
| `≤ -12%` | 찐바닥 — E/F 차단 해제 |

- 한 번 차단되면 히스테리시스로 바로 풀리지 않고, `-2.5%` 이상 회복 또는 `-12%` 이하 극단 구간이 나와야 해제된다.
- B는 공포장 반등 전략이므로 하락장 필터를 적용하지 않는다.

### 나스닥 상단 과열 매수 차단
- 적용 대상: 전략 전체 공통
- 기준:
  - 급락 후 회복장: QQQ 이격도 `> +18%`
  - 비회복장/고점 횡보장: QQQ 이격도 `> +9%`
- 신규 진입 / 추가 진입 차단
- 기존 보유분은 즉시 강제청산하지 않고, 별도 매도 조건으로 관리한다.

## 2. 매수 그룹

우선순위는 **A > B > C > D > E > F**다.

동시 충족 시 더 강한 확신의 시그널 하나만 선택해 중복 진입과 해석 충돌을 줄인다. 단, 그룹 조건을 만족해도 동적 공통 상단 차단(회복장 `QQQ > +18%`, 비회복장 `QQQ > +9%`), 이벤트, 고점 청산 상태가 있으면 신규/추가진입은 불가하다.

### A그룹 — 모멘텀 재가속
- MA200 위
- MACD 골든크로스
- 종가 `%B > 80`
- `RSI > 70`
- QQQ 이격도 `≥ -3%`

의미:
- 이미 강한 추세가 다시 가속되는 구간만 잡는다.

### B그룹 — 공황 저점
- MA200 아래
- `VIX ≥ 30`
- `RSI < 35` 또는 `CCI < -150`
- LR 추세선 기울기 `> 0`
- 저가가 추세선 근처 터치
- 터치 기준: `저가 <= LR 추세선 × 1.05`

의미:
- 장기 추세는 약하지만, 공포 과매도와 저점 훼손 방어가 동시에 나오는 반등만 노린다.

### C그룹 — 스퀴즈 거래량 돌파
- MA200 위
- 전일 BB 스퀴즈
- 당일 BB 확장 시작
- 거래량 `≥ 1.5배`
- 종가 `%B > 55`
- `MACD hist > 0`
- QQQ 이격도 `≥ -3%`
- 전일 스퀴즈 기준: `BB폭 / 60일평균 < 0.45`
- 당일 확장 기준: `당일 BB폭 > 전일 BB폭 × 1.00`

의미:
- 눌려 있던 변동성이 거래량과 함께 위로 터지는 초입을 노린다.

### D그룹 — 상승 흐름 강화
- MA200 위
- `+DI > -DI`
- `ADX > 30`
- ADX 상승
- `MACD hist > 0`
- 종가 `%B 30~75`
- 내부 추세 조건상 QQQ 허용 범위는 `-3% ~ 13%`
- 실제 신규 진입은 공통 상단 차단 때문에 회복장은 `+18% 초과`, 비회복장은 `+9% 초과`에서 불가
- 청산은 `+12% 익절 / -25% 손절 / 30거래일 시간청산`

의미:
- 이미 진행 중인 추세가 무너지지 않고 더 단단해지는 구간만 추종하되, 오래 끌지 않고 짧게 수익을 회수한다.

### E그룹 — 스퀴즈 저점
- MA200 위
- BB 스퀴즈
- 저가 `%B ≤ 50`
- QQQ 정상 구간 또는 찐바닥 허용

의미:
- 장기 우상향 안에서 스퀴즈 눌림을 평균회귀 관점으로 매수한다.

### F그룹 — BB 극단 저점
- MA200 위
- 저가 `%B ≤ 5`
- QQQ 정상 구간 또는 찐바닥 허용

의미:
- 장기 우상향 안에서 과도한 단기 눌림만 아주 보수적으로 잡는다.

## 3. 매수 해제

### 공통
- `매수 → 관망`으로 바뀌어도 진입 정보(진입가, 진입일, 전략)는 유지한다.
- 매도 조건 추적은 계속한다.
- 공통 상단 차단(회복장 `QQQ > +18%`, 비회복장 `QQQ > +9%`)은 신규/추가진입 차단 조건이지, 기존 보유를 바로 `매도`로 바꾸는 조건은 아니다.

### 그룹별 핵심 해제 기준
- **A**: MA200 이탈 / MACD 꺾임 / 나스닥 필터 진입
- **B**: VIX 완화 / 과매도 해소 / 추세 기울기 꺾임
- **C**: MA200 이탈 / MACD 꺾임 / 나스닥 필터 진입
- **D**: MA200 이탈 / `+DI ≤ -DI` / MACD 꺾임 / 나스닥 필터 진입
- **E**: MA200 이탈 / 스퀴즈 해소 / 저가 `%B > 50` / 나스닥 필터 진입
- **F**: MA200 이탈 / 저가 `%B > 3` / 나스닥 필터 진입

### 결측 처리 원칙
- MACD, RSI, CCI, %B, BB폭 같은 데이터가 일시 결측이면 즉시 해제로 단정하지 않는다.
- 이유: 데이터 지연이나 수식 공백 때문에 정상 포지션을 불필요하게 꺾지 않기 위해서다.

## 4. 보유 중 관망에서 추가 매수 신호

보유 중 `관망`이 된 뒤 조건이 다시 좋아져도 기존 포지션을 `매수`로 복원하지 않는다. 보유 중 `관망 → 매수` 전환은 실제 추가매수 슬롯을 열 수 있을 때만 허용한다.

아래 조건을 모두 만족해야 한다.

- 현재가가 기존 진입가 대비 `-10% 이상` 추가 하락
- 관망 전환 후 `10거래일` 경과
- 현재 충족된 전략의 매수 조건 재충족
- 슬롯 한도 미도달

E/F 계열의 다른 전략 슬롯으로 넘어가는 교차 추가매수는 위 조건만으로는 부족하다.

- 같은 새 전략 신호가 `2회 연속` 확인되어야 한다.
- 현재 열린 슬롯이 0개이면 과거 매도 이력이 있어도 `신규 진입`으로 표시한다. `재진입 n회차` 표현은 현재 보유 슬롯이 있는 상태에서 추가 슬롯이 들어올 때만 사용한다.

즉, 기존 보유 전략이 다시 켜졌다는 이유만으로 `매수` 전환이나 메일 발송을 하지 않는다. 충분한 가격 눌림, 시간 경과, 현재 전략 조건이 모두 맞아 실제 추가 슬롯이 열릴 때만 `매수`로 본다. E/F 계열 교차 추가매수는 같은 신호를 한 번 더 확인해 단발성 휩소를 줄인다.

## 5. 매도 기준

### 1순위 — 나스닥 고점 청산
- **B / D** 보유분만 강제 매도
- **A / C / E / F** 보유분은 강제청산 제외
- 신규 진입 및 추가 진입은 전략과 무관하게 차단
- 메인 포지션과 추가매수 슬롯도 lot별 전략 기준으로 B/D만 청산

기준:
- 회복장: QQQ 이격도 `> +22%`, 또는 `> +18%` + RSI 과열 둔화 + QQQ MACD Histogram 2일 연속 둔화
- 비회복장: QQQ 이격도 `> +16%` + RSI 과열 둔화, 또는 `> +14%` + RSI 과열 둔화 + QQQ MACD Histogram 2일 연속 둔화

### 2순위 — 그룹별 목표 수익
- **A / B / C**: `+20%` 도달 즉시 매도
- **D**: `+12%` 도달 즉시 매도
- **E / F**: `+20%` 도달 후 MACD 히스토그램 둔화전환 확인 후 매도

### 3순위 — 손절
- **D 제외 전 그룹 공통**: `-30%`
- **D**: `-25%`

### 4순위 — 시간 청산
- **전 그룹 공통**: `60거래일` 경과 + 수익 중이면 매도
- **D**: `30거래일` 경과 시 수익 여부와 무관하게 매도
- **D 제외 그룹**: `120거래일` 경과 시 수익 여부와 무관하게 매도

### E/F 매도 상세
- `+20%`를 찍었다고 바로 팔지 않는다.
- 목표 수익권 진입 후 MACD 히스토그램이 둔화전환될 때 매도한다.
- 그래도 안 나오면 최대 `5거래일` 대기 후 매도한다.

이유:
- E/F는 눌림/평균회귀 성격이 강해, 목표 수익권 진입 뒤 힘이 꺾이는지까지 보는 편이 더 맞다.
- A/B/C는 모멘텀 포착 전략이라 목표 수익 달성 시 바로 정리한다.
- D는 추세 추종이지만 속도형 운용을 위해 더 낮은 목표와 더 짧은 최대보유일을 쓴다.

## 6. 매도 후 쿨다운과 신규 진입 제한

- 매도 후 `48시간` 동안은 절대 신규 진입 불가
- 매도 후 `10거래일` 이내에는 매수 조건 재충족 + 현재가가 직전 매도가 대비 `-3% 이상` 더 낮아져야 신규 진입 허용
- `10거래일`이 지나면 `-3%` 조건은 해제되고, 매수 조건만 보면 된다. 이 경우도 현재 슬롯이 0개이면 메일 표시는 `신규 진입`이다.
- 단, 나스닥 고점 청산 상태면 계속 차단
- 단, 동적 공통 상단 차단 상태면 계속 차단

이유:
- 직전에 털린 자리로 바로 다시 들어가면 같은 실패를 반복하기 쉽다.
- 짧은 시간은 무조건 쉬고, 그다음에도 일정 기간은 더 싼 가격에서만 다시 들어간다.

## 7. Lot별 독립 청산

각 lot는 진입 전략 기준으로 따로 관리한다.

예:
- A로 먼저 매수
- 이후 B 조건으로 추가매수
- A 물량은 A 기준, B 물량은 B 기준으로 각각 청산 판단

평시:
- 전략별 lot 독립 청산
- A는 먼저 익절될 수 있고, B는 조건이 남으면 더 보유 가능

예외:
- 나스닥 고점 청산도 lot별 전략 예외를 적용한다.
- B/D lot는 강제청산하고, A/C/E/F lot는 유지한다.

## 8. 이벤트 기간

- 금리 결정, CPI, PCE, 네마녀의 날 등 주요 이벤트 당일에는 신규 진입을 막는다.
- 기존 보유 포지션은 매도 조건 충족 시 정상 매도한다.
- 이벤트 중 보유 포지션이 관망 상태가 되더라도, 추가 매수는 보류할 수 있다.

이유:
- 이벤트 당일 변동성은 방향성보다 잡음이 커서 신규 진입 품질이 급격히 떨어진다.

## 9. 실제 판단 순서

### Step 1 — 시장 필터 확인
- 나스닥 고점 청산 상태인가?
- 이벤트 있는가?
- 나스닥 하락장 필터 걸리는가?
- 동적 나스닥 상단 과열 차단 상태인가?

### Step 2 — 포지션 상태 확인
- 보유 중인가, 미보유인가?

### Step 3 — 미보유라면
- `A > B > C > D > E > F` 순으로 체크한다.
- 단, 상단 차단 / 이벤트 / 고점 청산 상태면 진입하지 않는다.

### Step 4 — 보유 중이라면
1. 매도 조건 먼저 확인
2. 아니면 보유 유지 조건 확인
3. 보유 중 `관망` 상태라면 추가매수 조건(`-10%`, `10거래일`, 현재 전략 조건, 슬롯 한도)을 확인
4. E/F 계열 교차 추가매수는 같은 새 전략 신호가 2회 연속인지 확인
5. 위 조건을 충족해 실제 추가 슬롯이 열릴 때만 `매수` 전환

---

## 상태 전환 요약
- `관망`: 진입 조건 미충족, 이벤트 차단, 하락장 필터 차단, 신규 진입 쿨다운, 고점 청산 차단
- `매수`: 미보유 신규 진입 조건 충족, 또는 보유 중 추가매수 조건 충족으로 실제 추가 슬롯 생성 가능
- `관망(보유 유지)`: 매수 조건은 깨졌지만 매도 조건은 아직 아님
- `매도`: 목표 수익, 손절, 시간 초과, B/D 고점 청산 강제 매도

## 한 줄 요약
- **A/C**: 강세장 모멘텀 추종
- **D**: 강세장 추세 추종이지만 `+12% / -25% / 30거래일` 속도형 청산
- **B**: 공포장 반등
- **E/F**: 눌림/저점 평균회귀
- 이벤트 당일은 신규/추가 진입 차단
- 공통 상단 차단은 회복장 `QQQ > +18%`, 비회복장 `QQQ > +9%`에서 전 전략 신규/추가진입 차단
- 나스닥 고점 청산은 회복장/비회복장별 동적 이격도 기준과 RSI 둔화, QQQ MACD 히스토그램 둔화로 판단
- E/F만 +20% 후 MACD 둔화전환 매도
- 평시 추가매수는 `-10% + 10거래일 + 현재 전략 조건 충족 + 필요 시 E/F 교차 2회 연속 신호` 확인 후 실제 슬롯이 열릴 때만 `매수` 전환
- 단, 고점 청산 때도 B/D lot만 강제청산하고 A/C/E/F lot는 유지
